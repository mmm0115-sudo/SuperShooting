# 灯火回廊 ― ASCENT

> ⚠️ **ベータ版です。** 仕様・バランスは今後変わります。不具合の報告歓迎。

軌道上に放棄された巨大ステーション「回廊（コリドー）」。その最深部に眠る、
消えかけた地上文明を再起動できる最後のコア。暴走した6層の自動防衛を、
使い捨ての単機「ランタン号」で下層から最上層へ突破する縦スクロール弾幕シューティング。
Mac でネイティブに動く **Java（Swing / Java2D）**製、ブラウザ不要。

## 起動方法
- **`StellarCascade.app`** をダブルクリック（タイトルは「灯火回廊」）。
  - 初回に開けない場合は **右クリック →「開く」→「開く」** を一度。
- 予備：**`灯火回廊を起動.command`** をダブルクリック。
- CLI：`java -jar 灯火回廊.jar`

## ソースからビルド
```bash
javac 灯火回廊.java
jar cfe 灯火回廊.jar StellarCascade *.class
cp 灯火回廊.jar StellarCascade.app/Contents/Resources/
java StellarCascade selftest   # 動作検証（TEST_OK 表示で正常）
```

## 操作
| 操作 | キー |
|------|------|
| 移動 | 方向キー / WASD |
| ショット | Z / Space |
| ボム | X（敵弾を消し大ダメージ・豪華な衝撃波） |
| 低速移動（当たり判定表示） | Shift |
| **残響リワインド（巻き戻し）** | **R / C** |
| ポーズ | P / Esc |
| サウンド ON/OFF | M |

メニューで **モード**・**難易度（Easy / Normal / Hard / Lunatic）** を ←→ で切替。

## 世界と登場AI（全6層）
各層は半壊した人格AIが管理。下層から順に：
1. **ゲート**（自機狙い）／2. **ブルーム**（全方位リング）／3. **ヴォルテクス**（螺旋）
／4. **グリッド**（レーザー・壁）／5. **エコー**（時間差・誘導）／6. **ルクス**（複合・総合）。
各層に **中ボス**＋**ボス**。ボスは **通常↔スペル交互**の手作り弾幕（通常は名前非表示）で、
スペルカードは制限時間つき・ノーミス/ノーボム撃破でボーナス。

## ランタン号（自機）
使い捨ての単機。機首の橙ランプは残機が減るほど速く・赤く点滅する。
灯火（ともしび）の型を3種から選択：
- **常火**（標準）／**疾火**（高速・小当たり判定）／**業火**（低速・高火力）
- ショット：**散灯**（拡散）／**束灯**（集束）／**追灯**（誘導）

## 残響リワインド
第5層エコーの「一度鳴った音は消えない＝残響」をシステム化。**R/C** で直前の状態へ巻き戻し
（難易度別に最大秒数・回数・クールタイム）。使うと **残響使用（ECHO）** フラグが立ち、
**正規踏破（ノーコンテ）扱いにならない**。画面隅に `ECHO` を表示。

## 競技モード・自己ベスト
クリア時の **RTAタイム** と **スコア** を、**難易度×モード×残響有無** のカテゴリ別に
永続保存（`~/.tomoshibi_records`）。メニューの **「記録」** で一覧（★=正規踏破）。

## モード
- **本編** … 全6層クリアを目指す。
- **回廊無限** … 段階的に激化する弾を流し続ける。**生存時間**を競う。
- **灯火・制限時間** … 制限時間内の到達度。
- **無音** … BGM/SEを切ったチャレンジ。
- **弾幕鑑賞** … 各ボスの弾幕を**無敵**で観賞・避け練習。

## パフォーマンス
固定解像度のオフスクリーンに1回描き、1枚で拡大表示。Retina/フルスクリーンでも描画コスト一定。
サイドバー下部に **FPS**（緑=快適/黄=低下/赤=処理落ち）。
