# STELLAR CASCADE 🚀（Java 版）

> ⚠️ **これはベータ版です。**
> 遊べる状態ですが、動作が不安定だったり、仕様・バランスが今後変わる可能性があります。
> 不具合や気づいた点があれば報告いただけると助かります。

Mac でネイティブなウィンドウとして動く、**6面制の弾幕シューティングゲーム**です。
**Java（Swing / Java2D）**で書かれており、ブラウザは不要です。

## 起動方法

### いちばん簡単（アプリとして起動）
Finder で **`StellarCascade.app`** をダブルクリック。独立したウィンドウで起動します。

> 初回だけ macOS のセキュリティで開けない場合は、`StellarCascade.app` を
> **右クリック →「開く」→「開く」** を一度選べば、以降はダブルクリックで起動できます。

### 予備の方法
`StellarCascade を起動.command` をダブルクリックしても起動できます。

### コマンドラインから
```bash
java -jar StellarCascade.jar
```

## ソースからビルドし直す
```bash
javac StellarCascade.java
jar cfe StellarCascade.jar StellarCascade *.class
cp StellarCascade.jar StellarCascade.app/Contents/Resources/
```

動作確認（ウィンドウを開かずロジック＋描画を検証）:
```bash
java StellarCascade selftest      # → TEST_OK ... と表示されれば正常
```

## 操作方法
| 操作 | キー |
|------|------|
| 移動 | 方向キー / W A S D |
| ショット | Z / Space（押しっぱなしで連射） |
| ボム | X（画面の敵弾を消して大ダメージ） |
| 低速移動（当たり判定を表示） | Shift |
| ポーズ | P / Esc（ポーズ中に Q でタイトルへ） |
| サウンド ON/OFF | M |
| 決定 | Z / Enter |

メニューで **難易度（EASY / NORMAL / HARD）** を ← → で切り替えられます。

## 機体・装備の選択
「ゲームスタート」を選ぶと **機体・装備 選択画面** に入ります（↑↓で項目、←→で変更、Zで決定）。

**機体（3種）**
- **VANGUARD**：標準・バランス型
- **ZEPHYR**：高速・当たり判定が小さい（火力やや低め）
- **AEGIS**：低速・高火力

**ショット（3種）**
- **WIDE 拡散**：広範囲をカバー
- **FORWARD 集束**：正面に集中・高火力
- **HOMING 誘導**：弾が敵を自動追尾

## アイテム自動回収（POC）
画面に点線の **POC ライン** があります。プレイヤーがそれより**上に行くと、画面中のアイテムが
一気に自動回収**されます（パワー・得点を取りこぼしません）。

## 難易度について
EASY と NORMAL/HARD の差が大きかったため **EASY を底上げ**して差を緩和しました。
- **EASY**：やや緩め。残機5・ボム4。
- **NORMAL**：標準。
- **HARD**：弾が速く多く、ボスは攻撃を重ねてきます。

敵の **HP を増量**し、**ステージが進むほど硬く**なります。ウェーブ数・敵の数も増やしてボリュームアップしました。

## ゲーム内容
- **メニュー画面**からスタート、全 **6 ステージ**（各ステージ末にボス戦）。
- 6 体の個性的なボス（多段階フェーズ制・HPバー・WARNING演出）。
- **赤 P** パワーアップ（最大 Lv.5）、**黄 •** 得点、**青 B** ボム増加、**緑 1** 残機増加。
- 効果音・簡易BGMを内蔵（外部ファイル不要、自前シンセで生成）。

## いちばんの特徴：二度と同じものが出ない
敵キャラクターと弾幕はその場で**プロシージャル生成**され、さらに
**シグネチャ（特徴量）照合**で重複を排除しています。そのため
**1 プレイ中に「まったく同じ敵」「まったく同じ弾幕」が二度出現しない**ことを保証。
編隊内の個体すら微妙に変異させ、同一個体が並ばないようにしています。
クリア画面に、その回に出現した「固有の敵タイプ数・固有の弾幕数」を表示します。

## Windows 対応について（予定）
Windows 版にも対応する予定です。本体は **プラットフォームに依存しない Java（Swing / Java2D）** で
書かれているため、Java 17 以降が入った Windows でも、理屈の上では次のコマンドで起動できる見込みです。

```bash
java -jar StellarCascade.jar
```

ただし、**開発環境に Windows マシンがないため、現時点では Windows 上での動作確認ができていません。**
そのため Windows での起動・動作は**保証できません**（フォント表示・キー入力・サウンドなどに
プラットフォーム差が出る可能性があります）。

なお、`StellarCascade.app` と `StellarCascade を起動.command` は **macOS 専用**なので、
Windows では使えません。Windows で動かす場合は上記のコマンドライン起動を試してください。

> Windows で試してくださった方がいれば、動いたか／不具合が出たかを報告いただけると、
> 正式対応の助けになります。
